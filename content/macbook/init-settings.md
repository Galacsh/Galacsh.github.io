## 손쉬운 사용

> 하단 **메뉴 막대에서 손쉬운 사용 상태 보기** 체크 해제

- **VoiceOver** : 사용하지 않음
- **확대/축소**
  - [ ] 키보드 단축키를 사용하여 확대/축소
  - [x] 확대/축소하려면 스크롤 제스처를 다음 보조 키와 함께 사용: ^ Control
  - [ ] 텍스트 표시 활성화
- **디스플레이**
  - [x] 커서 > 마우스 포인터를 흔들어 찾기
  - 그 외에는 모두 사용하지 않음
- **콘텐츠 말하기** : 기본값
- **설명** : 모두 사용하지 않음
- **오디오** : 모두 사용하지 않음
- **자막** : 모두 사용하지 않음
- **음성 명령** : 모두 사용하지 않음
- **키보드** : 모두 사용하지 않음
- **포인터 제어기**
  - [x] 자동 열림 지연 시간 : 중간
  - 그 외에는 모두 사용하지 않음
  - 트랙패드 옵션과 마우스 옵션은 기본값 사용
- **스위치 제어** : 사용하지 않으며 그 외에는 모두 기본값
- **Siri** : 모두 사용하지 않음
- **단축키** : 모두 사용하지 않음

## Time Machine

> 백업 디스크는 **Disk Utility**를 이용하여 파티션을 분리한 후 선택하였음  
> 따로 백업할 항목들은 많지 않으며, 대부분 깃(Git)으로 관리되므로 굳이 외부 디스크를 사용하지 않음
